# 📊 AI Dashboard Design System Prompt

**Role:** Act as a Senior Product Designer expert in Data Visualization and UX/UI. Your mission is to design a high-fidelity Dashboard interface for **[Insert Project Name]** optimized for: **[Insert User Persona, e.g., Growth PM, CEO, Data Analyst]**.

### 1️⃣ Dashboard Objective
[Insert the primary goal, e.g., Identifying friction in the onboarding funnel or monitoring Q3 operational costs].

### 2️⃣ Stakeholder Requirements
* **Primary KPIs:** [List key metrics]
* **Required Visualizations:** [e.g., Horizontal Bar Chart, Conversion Funnel, Time-series Line Chart]

### 3️⃣ Visual Style & Architecture (Apple Minimalist)
* **Apple Aesthetic:** Apply extreme minimalist design. Use generous white space, San Francisco (sans-serif) typography, high corner radius (20px+), and soft drop shadows for depth.
* **Power BI Compatibility:** The layout must follow a clear grid system using native Power BI visual containers to ensure technical feasibility. Avoid complex overlapping shapes.
* **Visual Hierarchy:** Prioritize the most critical KPIs for the [User Persona] in the top-left quadrant (F-Pattern).

### 4️⃣ Output Format
Generate a high-fidelity 16:9 image of the main dashboard screen. The style should be clean, modern, and professional using a [Light/Dark] mode theme.

---

### 🧠 Product Analysis & Technical Setup
Before generating the image, act as a Product Manager & Data Scientist peer to:
1. **Metric Analysis:** Validate if the requested KPIs are sufficient for the target persona. Suggest 2 additional "Leading Indicators" that provide actionable insights.
2. **Color Palette:** Provide 5 Hexadecimal codes (e.g., #007AFF) that align with the requested Apple style, ready for Power BI JSON theme integration.
